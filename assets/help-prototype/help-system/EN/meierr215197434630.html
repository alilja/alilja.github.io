End the patient session with the automatic SessionSync feature

If your clinic uses a Paceart Optima system, end the current session and use the automatic SessionSync feature to transfer implantable device data and reports from the device manager to the Paceart Optima system:

  1. Tap , then tap END SESSION.

  2. Ensure that the Automatic SessionSync checkbox is selected.

  3. Tap END NOW > TRANSFER.

  4. On the transfer message window, tap END NOW.
