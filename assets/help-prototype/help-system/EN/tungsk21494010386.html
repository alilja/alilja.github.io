Configure the Initial Interrogation Report preferences

Enable the Initial Interrogation Report, then select the reports that you want to include in the Initial Interrogation Report:

  1. Tap , then tap SESSION > PREFERENCES > INITIAL REPORT.

  2. Complete the following actions:

    1. Select the Automatically generate initial interrogation report after first interrogation checkbox.

    2. Select the reports to include in the Initial Interrogation Report.

      Note:

      The Quick Look Report is always included in the Initial Interrogation Report.

    3. Tap OK.

To generate an Initial Interrogation Report for a patient session that is in progress, end and restart the patient session.
