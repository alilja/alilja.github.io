Perform an Underlying Rhythm Test

To evaluate the patient’s intrinsic heart rhythm by temporarily inhibiting the pacing output of the implanted device, use the Underlying Rhythm Test.

Caution:

While the Underlying Rhythm Test is in progress, patients are not receiving pacing support because the implanted device is temporarily programmed to a nonpacing mode. Pacing is inhibited as long as you press and hold the INHIBIT Press and Hold button. Carefully consider the implications of performing this test on pacemaker-dependent patients.

Notes:

  • If telemetry between the implantable device app and the implanted device is paused or lost during an Underlying Rhythm Test, the test stops and the implanted device parameters revert to permanently programmed values.

  • During system tests, tachyarrhythmia detection is suspended.

  1. Tap , then tap TESTS > UNDERLYING RHYTHM.

  2. Verify the permanent values for Mode and Lower Rate.

  3. To help avoid sudden changes in ventricular rate, consider lowering the programmed lower rate:

    1. Tap , then tap PARAMETERS > Lower Rate.

    2. Select an appropriate rate.

    3. To program any pending changes, tap PROGRAM.

    4. To return to the UNDERLYING RHYTHM screen, tap , then tap TESTS > UNDERLYING RHYTHM.

  4. Press and hold INHIBIT Press and Hold.

  5. Observe the display of the heart’s intrinsic rhythm.

    Note:

    Pacing is inhibited until you release the INHIBIT Press and Hold button.

  6. Release the INHIBIT Press and Hold button.

  7. If Lower Rate was changed before conducting the Underlying Rhythm Test, return to the PARAMETERS screen to return the rate to its original value.
