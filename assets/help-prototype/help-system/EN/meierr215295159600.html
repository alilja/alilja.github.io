Configure waveform strip preferences

To set clipping and artifacts options, or to set the default duration for new waveform strip highlights, configure waveform strip preferences:

  1. From the FROZEN STRIP window, tap .

  2. Use the following options:

    Option Description
    Clipping When ON, truncates the tops and bottoms of waveforms that have high amplitudes.
    Show Artifacts When ON, displays line boundaries at the beginning and end of each wave. This feature is also known as pacing artifact enhancement.
    HIGHLIGHT DURATION Allows you to set the default duration for all new waveform strip highlights.

  3. To save your preferences, tap OK.
