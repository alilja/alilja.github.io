Configure the Final Report preferences

Select the reports that you want to include in the Final Report:

  1. Tap , then tap SESSION > PREFERENCES > FINAL REPORT.

  2. Complete the following actions:

    1. Select the reports to include in the Final Report. If you are configuring Final Report preferences for the first time, select All Settings in the PARAMETERS section.

      Note:

      The Session Summary Report is always included in the Final Report.

    2. Tap OK.
